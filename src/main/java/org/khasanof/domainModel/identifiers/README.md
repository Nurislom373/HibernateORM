# Hibernate Identifiers

Identifiers in Hibernate represent the primary key of an entity. This implies the values are unique so that they can
identify a specific entity, that they aren't null and that they won't be modified.

Hibernate provides a few different ways to define identifiers. In this article, we'll review each method of mapping
entity ids using the library.
