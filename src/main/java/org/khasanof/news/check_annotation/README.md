# Database Level Checks

Hibernate `@Check` annotatsiyasini taklif qiladi, shunda biz ushbu annotatsiya bilan check larini database levelda 
yozishimiz mumkin bo'ladi.

Agar Tekshiruvdan o'tmagan taqdirda `ConstraintViolationException` tashlanadi.

[Link](https://docs.jboss.org/hibernate/orm/6.1/userguide/html_single/Hibernate_User_Guide.html#schema-generation-database-checks)
